SPARQL end point

The end point is located at http://lod.cedar-project.nl/cedar/sparql and can be queried using SPARQL 1.1. Use the editor below to enter you own query, the results are displayed just under it. If you don't know what to query for, try one of these sample queries:

  • Count the number of triples per graph
  • Source of the raw datasets
  • Find column headers according to their label
  • Get a list of the dimensions used by the harmonized dataset
  • Get a list of the headers used by the harmonized dataset
  • Evolution of the number of married women over the years
  • Get the number of sheets parsed as raw-data
  • Get a list of raw datasets and their location
  • List of top 100 most occurring header in all the raw dataset
  • Number of shoemakers for each censsus year for each province grouped by age
  • House count of type 2-72 over the years
  • Observations of the house 2-72 by years count

Mini-project queries:

  • Reconstructing hierarchical headers (max depth 3)
  • Reconstructing hierarchical headers (max depth 4)
  • Temporarily present males in a certain municipality
  • Working totals
  • Totals per municipality
  • Totals per municipality in several years, ordered by municipality

Error-detecting queries:

  • Observations with more than one occupation position
  • Observations with negative population counts
  • Observations with non-integer population counts
